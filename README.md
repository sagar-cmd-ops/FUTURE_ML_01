# 📊 Sales & Demand Forecasting for Businesses

Machine Learning Internship Project – Future Interns

🚀 Live App:https://futureml01-6ktpjvwvjhrrfmafpfjdmo.streamlit.app/

## 📌 Project Overview
This project builds a machine learning model to forecast future sales using historical business data.

## 📂 Dataset
Superstore Sales Dataset

## 🛠 Tools Used
- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib
- Power BI

## ⚙️ Project Workflow
1. Data Cleaning  
2. Feature Engineering  
3. Model Training  
4. Sales Prediction  
5. Visualization in Power BI

## 📈 Dashboard Visualizations
- Actual vs Predicted Sales Trend  
- Monthly Sales Trend  
- KPI Cards (Total Sales, Predicted Sales, Average Sales)

## 💡 Business Insights
- Sales forecasting helps businesses predict future demand.
- Monthly trends reveal seasonal patterns.
- Businesses can plan inventory using forecast predictions.
